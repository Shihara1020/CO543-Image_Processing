# CO543 Image Processing 

This repository contains **CO543 Image Processing** labs (Lab 1–Lab 8).

![Course](https://img.shields.io/badge/Course-CO543%20Image%20Processing-blue)
![Language](https://img.shields.io/badge/Language-Python%20%2F%20Jupyter-yellow)

## Labs

- **Lab 1**: Getting familiar with basic OpenCV functions and CoLab
- **Lab 2**: Operations on images, masking, intensity transformations (brightness, contrast, histogram)
- **Lab 3**: Spatial filtering (lowpass, highpass, Laplacian, Gaussian, median, mean)
- **Lab 4**: Segmentation
- **Lab 5**: Feature extraction
- **Lab 6**: Pattern recognition
- **Lab 7**: CV / CNN
- **Lab 8**: Fourier transform
